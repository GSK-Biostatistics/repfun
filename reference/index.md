# Package index

## Reporting

- [`ru_freq()`](https://gsk-biostatistics.github.io/repfun/reference/ru_freq.md)
  : Create Percentage based on Numerator and Denominator Data
- [`ru_sumstats()`](https://gsk-biostatistics.github.io/repfun/reference/ru_sumstats.md)
  : Calculate Descriptive Statistics
- [`ru_list()`](https://gsk-biostatistics.github.io/repfun/reference/ru_list.md)
  : R function to mimic the SAS macro %tu_list.

## Utility

- [`rs_setup()`](https://gsk-biostatistics.github.io/repfun/reference/rs_setup.md)
  : Pass values to setup function and the global environment will be
  arranged for use reporting tools.
- [`ru_addbignvar()`](https://gsk-biostatistics.github.io/repfun/reference/ru_addbignvar.md)
  : Assign Big N to Data Frame.
- [`ru_addpage()`](https://gsk-biostatistics.github.io/repfun/reference/ru_addpage.md)
  : Add a Page Number Column to an Existing Dataframe
- [`ru_addsupp()`](https://gsk-biostatistics.github.io/repfun/reference/ru_addsupp.md)
  : Cobmine an SDTM domain with its corresponding Supplemental data set
- [`ru_align()`](https://gsk-biostatistics.github.io/repfun/reference/ru_align.md)
  : Align Columns for Reporting
- [`ru_contents()`](https://gsk-biostatistics.github.io/repfun/reference/ru_contents.md)
  : Generate proc contents of data frame similar to that generated by
  SAS
- [`ru_data2codelist()`](https://gsk-biostatistics.github.io/repfun/reference/ru_data2codelist.md)
  : Return a list with codelist, code and label based on input codelist
  dataset.
- [`ru_datacompare()`](https://gsk-biostatistics.github.io/repfun/reference/ru_datacompare.md)
  : Compare 2 data frames and report differences.
- [`ru_datetime()`](https://gsk-biostatistics.github.io/repfun/reference/ru_datetime.md)
  : Add numeric datetimes to data frame that only has character
  versions.
- [`ru_denorm()`](https://gsk-biostatistics.github.io/repfun/reference/ru_denorm.md)
  : Transpose a Data Frame
- [`ru_expvarlist()`](https://gsk-biostatistics.github.io/repfun/reference/ru_expvarlist.md)
  : Expand SAS Style Variable/Column List
- [`ru_fillcodedcode()`](https://gsk-biostatistics.github.io/repfun/reference/ru_fillcodedcode.md)
  : Fill missing code/decode records
- [`ru_fillna()`](https://gsk-biostatistics.github.io/repfun/reference/ru_fillna.md)
  : Fill NA values with specified values or zeros/blanks by default.
- [`ru_getdata()`](https://gsk-biostatistics.github.io/repfun/reference/ru_getdata.md)
  : Assign Big N to Data Frame.
- [`ru_groupbyvars()`](https://gsk-biostatistics.github.io/repfun/reference/ru_groupbyvars.md)
  : Modify groupbyvars by adding or removing decode vars.
- [`ru_labels()`](https://gsk-biostatistics.github.io/repfun/reference/ru_labels.md)
  : Assign labels to variables in a data frame
- [`ru_libname()`](https://gsk-biostatistics.github.io/repfun/reference/ru_libname.md)
  : Automate retrieval of data sets/frames in specified folder.
- [`ru_load_library()`](https://gsk-biostatistics.github.io/repfun/reference/ru_load_library.md)
  : Load a list of libraries
- [`ru_setdata()`](https://gsk-biostatistics.github.io/repfun/reference/ru_setdata.md)
  : Append data sets even when variables do not match
- [`ru_stackvar()`](https://gsk-biostatistics.github.io/repfun/reference/ru_stackvar.md)
  : Stack Columns of a Dataframe into New Column
- [`ru_width_rtf()`](https://gsk-biostatistics.github.io/repfun/reference/ru_width_rtf.md)
  : Create a List of Relative Widths of Columns for RTF Outputs
- [`setpath()`](https://gsk-biostatistics.github.io/repfun/reference/setpath.md)
  : Set the current working directory.
- [`rem_pg_nums()`](https://gsk-biostatistics.github.io/repfun/reference/rem_pg_nums.md)
  : Remove floating page numbers from RTF files.
- [`copydata()`](https://gsk-biostatistics.github.io/repfun/reference/copydata.md)
  : Copy package data to the directory specified.
